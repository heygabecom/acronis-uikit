## API Reference

{/* No exported Props interface â€” NavigationMenu sub-components wrap Radix UI Navigation Menu primitives */}

This component re-exports primitives from Radix UI Navigation Menu. All sub-components (`NavigationMenu`, `NavigationMenuList`, `NavigationMenuItem`, `NavigationMenuTrigger`, `NavigationMenuContent`, `NavigationMenuLink`, `NavigationMenuViewport`, `NavigationMenuIndicator`) extend their corresponding Radix UI primitive props. Refer to the [Radix UI docs](https://www.radix-ui.com/primitives/docs/components/navigation-menu) for the full API.
