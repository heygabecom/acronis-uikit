## API Reference

{/* No exported Props interface — Progress extends Base UI Progress.Root props */}

The `Progress` component wraps Base UI Progress primitives and extends `React.ComponentPropsWithoutRef<typeof ProgressPrimitive.Root>`. Pass `value` as a number (0-100) or `null` for indeterminate state (displays an animated progress indicator).
