## API Reference

### DialogContent

<AutoTypeTable
  path="../../packages/ui-legacy/src/components/ui/dialog.docs.ts"
  name="DialogContentProps"
/>

The Dialog is built on Base UI Dialog primitives. `Dialog` wraps `Dialog.Root`, `DialogTrigger` wraps `Dialog.Trigger`. `DialogHeader`, `DialogFooter`, and `DialogBody` accept standard HTML `div` attributes. `DialogTitle` and `DialogDescription` extend their respective Base UI primitive props.
