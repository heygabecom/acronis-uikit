## API Reference

{/* No exported Props interface — Form sub-components use react-hook-form and native HTML element props */}

The `Form` component is a re-export of `react-hook-form`'s `FormProvider`. `FormField` accepts `ControllerProps` from `react-hook-form`. Sub-components `FormItem`, `FormLabel`, `FormControl`, `FormDescription`, and `FormMessage` extend their native HTML element props and connect to form context via the `useFormField` hook.
