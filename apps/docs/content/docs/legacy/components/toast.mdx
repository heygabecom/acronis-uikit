## Overview

This project uses [Sonner](/docs/components/sonner) as the toast notification system. There is no separate `Toast` component -- all toast functionality is provided through the `Toaster` component and the `toast()` function from Sonner.

## Usage

```tsx
import { Toaster } from '@acronis-platform/shadcn-uikit';
import { toast } from 'sonner';
```

Add `<Toaster />` to your root layout, then trigger toasts anywhere:

```tsx
toast('Event has been created');
toast.success('Profile updated');
toast.error('Something went wrong');
toast.warning('Rate limit approaching');
```

See the [Sonner documentation page](/docs/components/sonner) for full examples and API reference.
