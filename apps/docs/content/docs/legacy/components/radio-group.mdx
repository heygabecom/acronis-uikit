## API Reference

{/* No exported Props interface — RadioGroup and RadioGroupItem extend Base UI primitive props */}

The `RadioGroup` extends Base UI `RadioGroup` primitive props and accepts `value`, `defaultValue`, and `onValueChange` for controlled usage. `RadioGroupItem` extends `Radio.Root` props and accepts a `value` string to identify each option.
