## API Reference

{/* No exported Props interface — Tabs sub-components wrap Base UI Tabs primitives */}

The `Tabs` component wraps Base UI Tabs primitives. The `Tabs` root accepts `defaultValue` and `value`/`onValueChange` for controlled usage. `TabsList` extends `Tabs.List` props, `TabsTrigger` maps to `Tabs.Tab`, and `TabsContent` maps to `Tabs.Panel`.
