## API Reference

{/* No exported Props interface — Separator extends Base UI Separator primitive props */}

The `Separator` wraps the Base UI Separator primitive and accepts `orientation` (`"horizontal"` | `"vertical"`, default `"horizontal"`) plus a `decorative` boolean prop and all standard HTML `div` attributes.
