`@acronis-platform/ui-react` is the next-generation Acronis React component
library: a [Base UI](https://base-ui.com/) implementation themed entirely by
generated design tokens. Components are accessible by default, fully typed, and
styled with Tailwind CSS v4 over the `--ui-*` custom properties from
`@acronis-platform/tokens-pd`.

## Features

- **Base UI primitives** — keyboard navigation, focus management, and ARIA come
  from `@base-ui/react`, a direct dependency
- **Token-driven theming** — colors, typography, and component metrics resolve to
  `--ui-*` tokens generated from the design system; light/dark via `light-dark()`
- **Multiple brands** — `acronis` by default, with brand overrides layered by
  import order
- **TypeScript-first** — every component is fully typed
- **Tree-shakeable** — import only the components you use
- **Acronis icon set** — companion `@acronis-platform/icons-react` package with
  per-size stroke scaling

## Quick start

```bash
pnpm add @acronis-platform/ui-react react react-dom
```

```tsx
import '@acronis-platform/ui-react/styles';
import { Button } from '@acronis-platform/ui-react';

export function App() {
  return <Button variant="default">Get started</Button>;
}
```
