## Usage

```tsx
import { Input } from '@acronis-platform/ui-react';
```

`Input` is a styled native `<input>` themed by the `--ui-input-text-*` tokens.
It accepts every standard `<input>` attribute. The error state is driven by
`aria-invalid`. Label, description, and error message are composed by the
consumer.

## Examples


<DemoReact>
  <InputDemo />
</DemoReact>

A basic input:

```tsx
<Input type="text" placeholder="Your name" />
```

Different input types:

```tsx
<Input type="email" placeholder="you@example.com" />
<Input type="password" placeholder="Password" />
```

Error and disabled states:

```tsx
<Input aria-invalid placeholder="Invalid value" />
<Input disabled placeholder="Disabled" />
```

## API Reference

`Input` is a themed native `<input>`. It accepts all standard HTML `<input>`
attributes (`type`, `value`, `onChange`, `placeholder`, `disabled`, …) and
forwards its ref.
