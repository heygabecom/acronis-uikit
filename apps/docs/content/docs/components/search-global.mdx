## Usage

```tsx
import { SearchGlobal } from '@acronis-platform/ui-react';
```

`SearchGlobal` is the app-level global search field — a leading magnifier, the
input, and an optional trailing keyboard-shortcut hint. It paints a gradient
border via the `--ui-input-search-*` tokens and accepts every standard
`<input>` attribute.

## Examples


<DemoReact>
  <SearchGlobalDemo />
</DemoReact>

A global search field with the default shortcut hint:

```tsx
<SearchGlobal placeholder="Search everything…" />
```

Customize or hide the shortcut hint:

```tsx
<SearchGlobal placeholder="Search…" shortcut="⌘K" />
<SearchGlobal placeholder="Search…" shortcut={null} />
```

Controlled:

```tsx
const [query, setQuery] = React.useState('');

<SearchGlobal
  value={query}
  onChange={(e) => setQuery(e.target.value)}
  placeholder="Search everything…"
/>
```
