## Usage

```tsx
import { Search } from '@acronis-platform/ui-react';
```

`Search` is a search box — a leading magnifier icon, the input, and a clear (×)
button that appears while there is a value. It accepts every standard `<input>`
attribute plus an `onClear` callback. Themed by the `--ui-input-search-*`
tokens.

## Examples


<DemoReact>
  <SearchDemo />
</DemoReact>

A basic search box:

```tsx
<Search placeholder="Search…" />
```

Controlled, with a clear handler:

```tsx
const [query, setQuery] = React.useState('');

<Search
  value={query}
  onChange={(e) => setQuery(e.target.value)}
  onClear={() => setQuery('')}
  placeholder="Search reports"
/>
```

Disabled:

```tsx
<Search placeholder="Search…" disabled />
```
